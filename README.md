# SW 유해성 검증 도구
본 스크립트는 소프트웨어(SW) 반입 전, 보안 체크리스트(엑셀) 작성을 돕기 위해 파일의 해시값과 전자서명 상태를 자동으로 추출하는 도구입니다.
검증결과(txt)를 활용하여 아래 링크 양식의 엑셀을 작성하고 txt와 함께 제출해주세요.
https://docs.google.com/spreadsheets/d/1liPC7i6VgYWtiANkSjFFBfXtGZZ04baN4-TfnkH2TNU

## 주요 기능
- **해시 추출**: 현재 폴더 및 하위 폴더의 모든 파일에 대해 SHA-256 해시값을 계산합니다.
- **전자서명 검증**: 실행 파일(EXE, DLL 등)의 디지털 서명을 확인하고, 서명자 정보를 추출합니다.
- **엑셀 호환성**: 검증 결과(Status)를 엑셀 양식의 드롭다운 항목(`Valid(정상)`, `Invalid(경고)`, `N/A(없음)`)에 맞춰 자동 변환합니다.

## 사용 방법
1. 검증하고자 하는 SW 파일들이 있는 폴더로 `SWvalidator.ps1` 파일을 복사합니다.
(모든 파일을 하나의 폴더에 두세요)
2. PowerShell 콘솔에서 스크립트를 실행합니다. 
(또는 스크립트 우클릭 후 "PowerShell에서 실행" 선택)
   ```powershell
   .\SWvalidator.ps1
   ```
3. 검사가 완료되면 `SW_Verification_Report_YYYYMMDD_HHMMSS.txt` 파일이 생성됩니다.
4. 생성된 텍스트 파일의 내용을 복사하여 **엑셀 체크리스트**에 활용할 수 있습니다.

## 주의 사항
- **폴더 감지 시 중단**: 스크립트 실행 경로 내에 하위 폴더가 존재하는 경우, 압축 제출이 원칙이므로 **오류 메시지를 출력하고 즉시 중단**됩니다. 
  - 하위 폴더가 있다면 반드시 `.zip` 등으로 압축한 뒤 스크립트를 다시 실행하십시오.
- **압축 파일**: `.zip`, `.7z` 등의 압축 파일은 해시값만 계산하며 전자서명은 `N/A(없음)`으로 처리됩니다.

## 결과물 예시
| 파일명 | 해시값(SHA-256) | 전자서명(Status) | 서명자(Signer) |
| :--- | :--- | :--- | :--- |
| app.exe | 9051BD81... | Valid(정상) | Google LLC |
| data.zip | 3C65DF2B... | N/A(없음) | N/A(없음) |
